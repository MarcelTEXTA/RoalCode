Documentation RoalCode

RoalCode est un langage de programmation interprété, typé statiquement et orienté performance, conçu pour le développement d’applications lourdes, professionnelles et maintenables.

Il privilégie la lisibilité du code, une gestion mémoire explicite et une programmation orientée objet moderne.

Caractéristiques

  • Langage interprété
  • Typage statique obligatoire
  • Gestion mémoire avancée via memoRoal
  • Programmation orientée objet complète
  • Exceptions natives
  • Système de modules extensible
  • Conçu pour des applications lourdes

Syntaxe générale

La syntaxe de RoalCode est volontairement explicite et proche de C#/Java, tout en supprimant les éléments inutiles pour améliorer la lisibilité. 

// Déclarations
int a = 10
float b = 2.5
string message = "Hello RoalCode"
bool active = true

Types primitifs

  • int — nombres entiers
  • float — nombres décimaux
  • string — chaînes de caractères
  • bool — valeurs booléennes
  • void — absence de valeur de retour

Structures de données

Listes

Les listes sont typées et indexées à partir de 0.

int[] numbers = [1, 2, 3, 4]
string[] cities = ["Paris", "Lyon"]

Dictionnaires

map<string,int> ages = {
    "Alice": 20,
    "Bob": 25
}

Conditions

RoalCode utilise des opérateurs logiques explicites.

if (value > 0) {
    console.print("Positif")
} else if (value == 0) {
    console.print("Zéro")
} else {
    console.print("Négatif")
}

Opérateurs logiques :

  • and
  • or
  • not

Boucles

Boucle for

for (int i = 0; i < 5; i = i + 1) {
    console.print(i)
}

Boucle for each

for each city in cities {
    console.print(city)
}

Fonctions

Les fonctions possèdent un type de retour obligatoire. 

int square(int x) {
    return x * x
}
void greet(string name) {
    console.print("Bonjour " + name)
}

Programmation orientée objet

Déclaration de classe

public class Person {
    string name
    int age

    void greet() {
        console.print("Bonjour, je suis " + name)
    }
}

Héritage

private class Animal : Person {
    string species
}

Création d’objet

Person p = new Person()
p.name = "Alice"
p.age = 20
p.greet()

Gestion des erreurs

RoalCode dispose d’un système d’exceptions typées. 

try {
    int result = 10 / value
} catch (DivisionByZeroError e) {
    console.print("Division par zéro")
} catch (Exception e) {
    console.print(e.message)
} finally {
    console.print("Fin du traitement")
}
